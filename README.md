# Pytho# Python Security Tools
A collection of custom Python scripts for system auditing and security monitoring on Linux.

## 🛠 System Auditor (`auditor.py`)
This script automates basic security checks on a Linux machine.

### Features:
* Lists all active TCP/UDP listening ports.
* Identifies the top 5 memory-consuming processes.
* Generates an automated timestamped report (`audit_results.txt`).

### How to use:
1. Run `python3 auditor.py` in the terminal.
